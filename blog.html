Question-1: What are the different ways to select an element in the DOM?

There are various ways to select an element in the DOM. Such as:
getElementsByTagName("tagName"); select all the elements of mentioned tag name.
getElementById("id") select a element with specific id.
getElementsByClassName("className")select all the element which had mentioned className.
querySelector(".class/#id/tagName"); Select the first element which had mentioned value.
querySelectorAll(".class/#id/tagName");Select all the elements which had mentioned value.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Both three are useful for accessing or modifying the DOM element. But there some key differences between them. Such as:
element.innerHTML Through this we can access or modify all the HTML code or text or comment as well as within it.
element.innerText Through this we can access or modify the visible innerText of that element
element.textContent Through this we can access or modify the visible and invisible (like comments) innerText of that element

Question-3: What is event delegation in the DOM?

Event delegation is a process to handle event in JavaScript. Actually when we add a event listener in a parent element instead of child element called event delegation. And we get several benefit from this process like when adding dynamic child element or load from api it works perfectly. And also help us to avoid code redundancy.

Question-4: What is event bubbling in the DOM?

Event bubbling is a concept in js DOM. it occurs when we clicked in an event it's triggered towards parent until find root element and it's called event bubbling. It's a JavaScript default behavior.
